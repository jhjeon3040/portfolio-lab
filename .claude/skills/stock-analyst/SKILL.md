# Wall Street Top 10 Stock Analyst

종목명 또는 티커를 받아 월가 10대 기관의 분석 방법론을 적용한 종합 리포트를 생성한다.

## 사용법

```
/stock-analyst {종목명 또는 티커}
```

예시:
- `/stock-analyst AAPL`
- `/stock-analyst 삼성전자`
- `/stock-analyst TSLA 기술적분석만` (특정 분석만 요청 가능)

## 실행 절차

### Step 1: 종목 정보 수집

WebSearch와 WebFetch를 활용하여 해당 종목의 최신 데이터를 수집한다:
- 현재 주가, 시가총액, 52주 최고/최저
- 최근 실적 (매출, 영업이익, 순이익)
- 주요 재무 비율 (P/E, P/B, ROE, 부채비율 등)
- 최근 뉴스 및 애널리스트 컨센서스

**데이터 소스 우선순위:**
1. Yahoo Finance / Google Finance (글로벌)
2. 네이버 금융 / KRX (한국 종목)
3. SEC EDGAR / DART (공시)

### Step 2: 10가지 분석 수행

수집된 데이터를 기반으로 아래 10가지 분석을 **병렬로** 수행한다. 사용자가 특정 분석만 요청한 경우 해당 분석만 수행.

## 분석 프레임워크

### 1. 골드만삭스급 종목 스크리닝

기관 투자자용 리서치 리포트 형식으로 출력.

| 항목 | 분석 내용 |
|------|-----------|
| P/E | 현재 vs 업종 평균 vs 5년 평균 |
| 매출 성장률 | 최근 3년 YoY + 향후 예상 |
| 부채비율 | D/E ratio, 이자보상배율 |
| 배당 수익률 | 현재 수익률, 배당성향 |
| 경쟁 해자 | 브랜드, 네트워크효과, 전환비용, 규모의 경제 |
| 리스크 등급 | Low / Medium / High + 근거 |

**출력 형식:** 투자의견(매수/중립/매도) + 목표가 범위 + 핵심 투자포인트 3개

### 2. 모건스탠리 DCF 가치평가

내재가치 vs 시장가격 비교.

- **매출 예측:** 향후 5년 매출 성장 시나리오 (낙관/기본/비관)
- **영업이익률:** 과거 추세 + 향후 전망
- **WACC:** 무위험이자율, 베타, 시장프리미엄, 부채비용으로 산출
- **잔존가치:** 영구성장률 모델
- **민감도 분석표:** WACC(행) x 영구성장률(열) 매트릭스

```
| 내재가치 | 영구성장률 2% | 2.5% | 3% |
|----------|-------------|------|-----|
| WACC 8%  |     $XXX    | $XXX | $XXX|
| WACC 9%  |     $XXX    | $XXX | $XXX|
| WACC 10% |     $XXX    | $XXX | $XXX|
```

**결론:** 현재가 대비 상승/하락 여력 (%)

### 3. 브릿지워터 리스크 분석

포트폴리오 관점의 리스크 관리 리포트.

- **상관관계:** 주요 자산군/지수와의 상관계수
- **섹터 집중도:** 매출/이익의 사업부별 비중
- **지역 노출도:** 매출의 지역별 비중
- **스트레스 테스트:** 경기침체, 금리급등, 달러강세 시나리오별 영향
- **유동성 등급:** 일평균 거래대금 기준 (A~D)
- **헤지 전략:** 풋옵션, 인버스ETF, 페어트레이딩 등 제안

### 4. JP모건급 실적 분석

- **어닝 서프라이즈/미스:** 최근 8분기 실적 vs 컨센서스 패턴
- **사업부별 분석:** 각 사업부 매출/이익 기여도 + 성장률
- **옵션 시장 반응:** 실적 발표 전후 내재변동성 변화
- **변동성 패턴:** 실적 발표 후 +1일, +5일 평균 주가 변동
- **매매 전략:** 실적 시즌 대응 전략

### 5. 블랙록 포트폴리오 구성 모델

- **자산 배분:** 주식/채권/대체자산 비중 제안
- **ETF 추천:** 해당 종목 포함 또는 관련 구체적 ETF 3~5개
- **기대 수익률:** 1년/3년/5년 시나리오별
- **리밸런싱:** 분기별/반기별 리밸런싱 기준
- **세금 효율:** 세금 최적화 전략 (손실 수확 등)

### 6. 시타델급 기술적 분석

- **추세:** 단기/중기/장기 추세 방향 (상승/횡보/하락)
- **지지/저항선:** 주요 가격대 3개씩
- **이동평균선:** 20/50/200일 MA 위치 + 골든/데드크로스
- **모멘텀:** RSI(14), MACD, 스토캐스틱
- **거래량:** OBV 추세, 거래량 이상 징후
- **차트 패턴:** 현재 형성 중인 패턴 (삼각수렴, 헤드앤숄더 등)
- **피보나치:** 주요 되돌림/확장 레벨
- **진입/청산:** 구체적 매수가, 손절가, 목표가

### 7. 하버드 기금 배당 전략

- **배당 안정성 점수:** 1~10점 (연속배당연수, 배당성향 기반)
- **연속 배당 성장:** 몇 년 연속 배당 증가했는지
- **배당성향:** 순이익 대비 배당 비율 + 지속가능성
- **DRIP 시뮬레이션:** 배당 재투자 시 10년/20년 후 예상 가치
- **세금 영향:** 배당소득세 고려 실수익률
- **동종업계 배당주 15~20개:** 비교 테이블

### 8. 베인 경쟁사 분석

같은 섹터 5~7개 기업 비교.

+ **SWOT 분석** (강점/약점/기회/위협)

### 9. 르네상스 테크놀로지스 패턴 탐지

통계적 이상치 탐지.

- **계절적 패턴:** 월별/분기별 수익률 히트맵
- **요일별 수익률:** 특정 요일 편향
- **내부자 거래:** 최근 임원 매수/매도 신호
- **숏스퀴즈 가능성:** 공매도 비율, 대차잔고
- **이상 옵션 활동:** 비정상적 콜/풋 거래량
- **이벤트 드리븐:** 예정된 카탈리스트 (실적, FDA, 분할 등)

## Step 3: 종합 결론

모든 분석을 종합하여 최종 리포트를 작성한다:

```
## 종합 투자 의견

### 투자의견: [Strong Buy / Buy / Neutral / Sell / Strong Sell]
### 목표가: $XXX (현재가 대비 +XX%)
### 투자기간: [단기(3개월) / 중기(1년) / 장기(3년+)]

### 핵심 불(Bull) 케이스 (3개)
1. ...
2. ...
3. ...

### 핵심 베어(Bear) 케이스 (3개)
1. ...
2. ...
3. ...

### 매매 전략
- 진입가: $XXX
- 1차 목표: $XXX
- 2차 목표: $XXX
- 손절가: $XXX
- 포지션 사이즈: 포트폴리오의 X%

### 모니터링 포인트
- 다음 실적 발표일: YYYY-MM-DD
- 주요 이벤트: ...
- 리스크 트리거: ...
```

## 출력 규칙

1. **한국어로 출력** (한국 종목은 원화, 미국 종목은 달러 기준)
2. **데이터 기반:** 수치와 근거를 반드시 포함. 추측은 명시적으로 표시
3. **테이블 적극 활용:** 비교 데이터는 표로 정리
4. **면책 조항:** 리포트 마지막에 "본 분석은 AI 기반 참고자료이며 투자 권유가 아닙니다" 명시
5. **데이터 시점:** 사용한 데이터의 기준일 명시
6. **실시간 데이터 부재 시:** WebSearch로 최신 데이터 검색 후 분석. 검색 불가 시 가용 데이터 기준으로 분석하되 한계 명시

## 축약 모드

사용자가 특정 분석만 요청할 수 있다:

- `/stock-analyst AAPL dcf` → 2번 DCF만
- `/stock-analyst 삼성전자 기술적분석` → 6번만
- `/stock-analyst TSLA 리스크 매크로` → 3번 + 10번
- `/stock-analyst NVDA 요약` → 10가지 각 1줄 요약 + 종합결론만

키워드 매핑:
- `스크리닝`, `골드만` → 1번
- `dcf`, `가치평가`, `모건스탠리` → 2번
- `리스크`, `브릿지워터` → 3번
- `실적`, `JP모건`, `어닝` → 4번
- `포트폴리오`, `블랙록`, `etf` → 5번
- `기술적`, `차트`, `시타델` → 6번
- `배당`, `하버드` → 7번
- `경쟁사`, `베인`, `swot` → 8번
- `패턴`, `르네상스`, `퀀트` → 9번
- `매크로`, `맥킨지`, `거시` → 10번
- `요약`, `summary` → 전체 1줄 요약
