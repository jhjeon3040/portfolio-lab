# Portfolio Builder — 자동 포트폴리오 구성 시스템

매크로 분석 → 유망 테마/섹터 도출 → 정량 스크리닝 → 정성 평가 → 최종 포트폴리오 구성까지 전 과정을 자동 수행한다.

## 사용법

```
/portfolio-builder                          # 풀 자동 (매크로부터 종목까지)
/portfolio-builder AI반도체                  # 특정 테마 지정
/portfolio-builder 배당성장 $10000           # 투자금액 지정
/portfolio-builder 공격적 $50000 미국+한국    # 성향 + 금액 + 시장
```

### 파라미터 (전부 선택사항)

| 파라미터 | 설명 | 기본값 |
|---------|------|--------|
| 테마 | 특정 테마 지정 시 Step 2 스킵 | 자동 도출 |
| 금액 | 투자 총액 | 미지정 시 비중%만 출력 |
| 성향 | 공격적 / 균형 / 보수적 | 균형 |
| 시장 | 미국 / 한국 / 미국+한국 / 글로벌 | 미국 |
| 종목수 | 포트폴리오 종목 수 | 5~10개 |

---

## 실행 절차

### Step 1: 매크로 환경 스캔

WebSearch로 최신 매크로 데이터를 수집하여 현재 시장 국면을 판단한다.

**수집 항목:**

| 항목 | 소스 | 판단 기준 |
|------|------|----------|
| 금리 방향 | Fed dot plot, FOMC 성명 | 인하→성장주, 인상→가치주/배당주 |
| 인플레이션 | CPI, PCE | 상승→원자재/에너지, 하락→기술주 |
| GDP 성장률 | BEA, FOMC 전망 | 확장→경기민감주, 둔화→방어주 |
| 실업률 | BLS | 상승→자동화/효율화 테마 |
| 달러 인덱스 | DXY | 강세→미국내수, 약세→수출/신흥국 |
| 수익률 곡선 | 2Y-10Y 스프레드 | 역전→경기침체 경고 |
| VIX | CBOE | >25 공포, <15 탐욕 |
| 자금 흐름 | ETF 순유입/유출 | 돈이 실제로 어디로 가는지 |

**검색 쿼리 예시:**
- "US macro economic outlook {current month} {current year}"
- "Fed funds rate FOMC latest decision {current year}"
- "sector rotation ETF fund flows {current month} {current year}"
- "VIX index level today"

**출력: 시장 국면 판정**

```
## 현재 시장 국면: [확장 초기 / 확장 중기 / 확장 후기 / 수축기]

- 금리: X.XX% (방향: 인하/동결/인상)
- 인플레: X.X% (추세: 하락/안정/상승)
- GDP: X.X% (추세: 가속/안정/둔화)
- VIX: XX (심리: 탐욕/중립/공포)
- 자금흐름: [기술주 유입 / 방어주 유입 / 현금 이탈]
```

### Step 2: 유망 테마/섹터 도출

매크로 국면에 따라 유망 테마를 자동 매핑한다.

**국면별 유망 섹터 매트릭스:**

| 국면 | 1순위 | 2순위 | 3순위 | 회피 |
|------|-------|-------|-------|------|
| 확장 초기 (금리인하 시작) | 기술/성장주 | 소형주 | 부동산(REITs) | 유틸리티 |
| 확장 중기 (경기 호황) | 경기민감주 | 산업재 | 금융 | 필수소비재 |
| 확장 후기 (과열 징후) | 에너지 | 원자재 | 헬스케어 | 기술/성장주 |
| 수축기 (경기둔화) | 필수소비재 | 유틸리티 | 헬스케어 | 경기민감주 |

**추가: 구조적 메가트렌드 (국면 무관)**

WebSearch로 현재 시장에서 실제로 돈이 몰리는 테마를 탐색:

검색 쿼리:
- "hottest investment themes {current year}"
- "fastest growing sectors {current year} stock market"
- "ETF inflows by sector {current month} {current year}"

메가트렌드 후보 풀:
- AI 인프라 (GPU, 데이터센터, 전력, 냉각)
- AI 소프트웨어 (SaaS, 엔터프라이즈 AI, 에이전트)
- 사이버보안
- 클라우드 컴퓨팅
- 전기차/자율주행
- 바이오테크/GLP-1
- 우주항공/방위산업
- 에너지 전환 (원전, 태양광, ESS)
- 리쇼어링/제조업 회귀
- 디지털 결제/핀테크

**출력:**

```
## 유망 테마 TOP 5

| 순위 | 테마 | 근거 | 확신도 |
|------|------|------|--------|
| 1 | ... | ... | High/Med/Low |
| 2 | ... | ... | ... |
| 3 | ... | ... | ... |
| 4 | ... | ... | ... |
| 5 | ... | ... | ... |
```

### Step 3: 종목 스크리닝 (정량 필터)

각 테마별로 대표 종목을 WebSearch로 탐색 후 정량 필터를 적용한다.

**검색 쿼리:**
- "best {theme} stocks to buy {current year}"
- "top {sector} companies by market cap revenue growth"
- "{theme} ETF top holdings"

**필수 통과 필터 (Hard Filter):**

| 필터 | 공격적 | 균형 | 보수적 |
|------|--------|------|--------|
| 매출 성장률 | >15% | >10% | >5% |
| 영업이익률 | >10% | >15% | >20% |
| 부채비율 (D/E) | <2.0 | <1.0 | <0.5 |
| 시가총액 | >$5B | >$10B | >$50B |
| PEG | <3.0 | <2.0 | <1.5 |

**가산점 필터 (Soft Filter):**

| 항목 | 점수 | 확인 방법 |
|------|------|----------|
| 어닝 서프라이즈 3분기 연속 Beat | +2 | WebSearch 실적 이력 |
| 내부자 순매수 | +2 | 최근 3개월 임원 매수>매도 |
| 애널리스트 목표가 상향 추세 | +1 | 컨센서스 변화 |
| 배당 성장 10년+ | +1 | 배당 이력 |
| FCF 마진 > 20% | +1 | 현금 창출력 |
| 수주잔고/RPO 증가 | +2 | 미래 매출 가시성 |
| Wide Moat (전환비용/네트워크/규모) | +2 | 정성 판단 |

**각 테마별 상위 5~10개 종목을 점수순으로 정렬.**

---

### Step 4: 정성 평가 (해자 + 촉매 + 미스프라이싱)

스크리닝 통과 종목에 대해 3가지 질문으로 최종 평가:

**Q1. 해자 (5년 뒤에도 대체 불가능한가?)**

| 해자 유형 | 설명 | 예시 |
|----------|------|------|
| 전환비용 | 바꾸기 어려움 | MSFT Office, ADBE, SAP |
| 네트워크 효과 | 사용자↑ = 가치↑ | VISA, META, ABNB |
| 규모의 경제 | 경쟁자 진입 불가 | AMZN, TSMC, ASML |
| 무형자산 | 특허/브랜드/규제 | NVDA CUDA, LLY 약품 |
| 비용 우위 | 구조적 저비용 | COST, WMT |

**Q2. 촉매 (지금 사야 하는 이유)**
- 다음 분기 실적 서프라이즈 예상
- 신제품/서비스 출시 임박
- 규제 완화/승인
- 자사주 매입 발표
- 섹터 로테이션 수혜

**Q3. 미스프라이싱 (시장이 뭘 놓치고 있나)**
- P/E가 역사적 평균 대비 저평가
- 성장률 대비 밸류에이션 할인
- 사업부 가치 합산(SOTP) > 현재 시가총액
- 부정적 뉴스에 과잉 반응

---

### Step 5: 포트폴리오 구성

최종 선정된 종목으로 포트폴리오를 구성한다.

**배분 원칙:**

| 성향 | 핵심(확신 높음) | 성장(중간) | 탐색(고위험) | 방어/현금 |
|------|---------------|----------|------------|----------|
| 공격적 | 40% (2~3종목) | 35% (3~4종목) | 15% (2~3종목) | 10% |
| 균형 | 35% (2~3종목) | 30% (2~3종목) | 10% (1~2종목) | 25% |
| 보수적 | 30% (2~3종목) | 20% (2~3종목) | 5% (1종목) | 45% |

**분산 규칙:**
- 단일 종목 최대 비중: 15% (핵심 종목도)
- 단일 섹터 최대 비중: 35%
- 미국/해외 비중: 최소 80/20
- 상관관계 높은 종목 2개 이상 불가 (예: MSFT+GOOGL 둘 다 핵심 X)

## 출력 규칙

1. **한국어로 출력** (금액은 사용자 지정 통화, 미지정 시 달러)
2. **모든 수치에 출처/기준일 표시**
3. **테이블 적극 활용**
4. **면책 조항:** "본 포트폴리오는 AI 기반 참고자료이며 투자 권유가 아닙니다"
5. **WebSearch 필수:** 매크로, 테마, 종목 데이터 모두 실시간 검색 기반

## 한국 시장 모드

시장 파라미터가 `한국` 또는 `미국+한국`인 경우:

**추가 검색:**
- "코스피 코스닥 유망 섹터 {current year} {current month}"
- "한국 주식 테마주 수급 {current month}"
- "외국인 기관 순매수 종목 {current month}"

**한국 종목 스크리닝 소스:**
- 네이버 금융, KRX 정보데이터시스템, FnGuide

**한국 특수 필터:**
- 외국인/기관 수급 동향 (순매수 전환 종목 가산점)
- 공매도 잔고 비율
- 신용잔고 비율

## 축약 모드

```
/portfolio-builder 요약           → 테마 3개 + 종목 5개 한줄씩
/portfolio-builder 테마만          → Step 1~2만 (매크로 + 테마)
/portfolio-builder 스크리닝만 AI   → Step 3만 (AI 테마 종목 스크리닝)
```

## stock-analyst 연동

포트폴리오 구성 후 개별 종목 딥다이브가 필요하면:
- "다음 액션" 섹션에 `/stock-analyst {종목}` 명령어를 포함
- 사용자가 바로 실행할 수 있도록 안내
